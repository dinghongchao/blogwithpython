自学python后的第一个练习项目，参考廖雪峰的官网的python实战章节
如果想零基础学习python 请移步至 [廖雪峰官网](http://www.liaoxuefeng.com/)

> 2.项目结构

```
awesome-python-webapp/   <-- 根目录
|
+- backup/               <-- 备份目录
|
+- conf/                 <-- 配置文件
|
+- dist/                 <-- 打包目录
|
+- www/                  <-- Web目录，存放.py文件
|  |
|  +- static/            <-- 存放静态文件
|  |
|  +- templates/         <-- 存放模板文件
|
+- LICENSE               <-- 代码LICENSE
```
